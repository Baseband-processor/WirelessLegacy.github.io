Lorcon2 in Air::Legacy

It all started from this point: Lorcon2, a library written in C and designed to inject potential malicious packets into the network, the first version dates back to 2005, time of WEP, WPA2 had not yet been implemented and, after many years, still support for TKIP/CCMP/WPA/WPA2 has not been included.
